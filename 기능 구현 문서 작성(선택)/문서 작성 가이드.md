# 기능 구현 문서 작성 가이드

### 1.폴더명 가이드
-  년도_월_일 방식으로 폴더 제작해 주세요.  
   - EX) 26_03_19 <-- yymmdd방식 부탁 드립니다.  
   
### 2.파일명 가이드  
- 기능을 알아 볼 수 있도록 (기능 구현  내용.md) 방식으로 파일을 만들어주세요  
  - EX) 퀘스트 시스템.md
  - EX) 캐릭터 이동 로직.md

- 주의 해주시면 좋은 것들
  - EX) ```캐릭터 제작.md``` OR ```몬스터 제작.md ```<-- X 상세한 내용으로 작성 부탁 드립니다.


###  3. 문서 작성 가이드

기능 구현 전, 아래 형식으로 간단한 설계 문서를 작성한다.

---

### 📍 작성 구조

1. **기능 이름 작성**
2. **구현해야 할 기능 목록 작성**
3. **각 기능의 검증(확인) 항목 작성**

---

### 📍 작성 템플릿

```md
[기능 이름]

- 기능 설명 1
  - 확인 사항 1
  - 확인 사항 2

- 기능 설명 2
  - 확인 사항 1

- 기능 설명 3
  - 확인 사항 1
